# `cursor:values`

> A complete list of CSS `cursor` values

#### [*Check out the site to see them all in action!*][site]

---

Below is the complete list of all CSS `cursor` values.

#### General
* `auto`
* `default`
* `none`

#### Links & Status
* `context-menu`
* `help`
* `pointer`
* `progress`
* `wait`

#### Selection
* `cell`
* `crosshair`
* `text`
* `vertical-text`

#### Drag & Drop
* `alias`
* `copy`
* `move`
* `no-drop`
* `not-allowed`

#### Resize & Scrolling
* `all-scroll`
* `col-resize`
* `row-resize`
* `n-resize`
* `e-resize`
* `s-resize`
* `w-resize`
* `ne-resize`
* `nw-resize`
* `se-resize`
* `sw-resize`
* `ew-resize`
* `ns-resize`
* `nesw-resize`
* `nwse-resize`

#### Zoom
* `zoom-in`
* `zoom-out`

#### Grab
**NOTE:** the following two values require vendor prefixes
* `grab`
* `grabbing`
